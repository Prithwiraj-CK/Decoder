## Quick Start

### Vanilla HTML/CSS/JS Version

1. Open `index.html` in your browser
2. No build process required!

### React Version

1. Set up a React project:
```bash
npx create-react-app sol-decoder
cd sol-decoder
```

2. Copy the React files:
- Replace `src/App.js` with `App.jsx`
- Replace `src/App.css` with `App.css`
- Copy all files from `components/` to `src/components/`

3. Install dependencies (if needed):
```bash
npm install
```

4. Start the development server:
```bash
npm start
```

## Customization

### Colors
Edit CSS variables in `:root` to change the color scheme:
```css
:root {
    --primary: #00ff88;
    --secondary: #ff0080;
    --accent: #00d4ff;
    /* ... */
}
```
